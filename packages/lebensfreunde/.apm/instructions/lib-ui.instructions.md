# Lib: UI / Designsystem (Beispiel)

> Dummy-Scaffold — exemplarische konkrete Lib. Ergänzt angular-libs.

- Rein präsentational: Inputs/Outputs (Signals), keine HTTP-Calls, kein globaler State, keine Routing-Logik.
- Styling über Design-Tokens/CSS-Custom-Properties; keine hartkodierten Farben, Abstände oder Schriftgrößen.
- Jede Komponente accessibility-konform: Rollen, Labels, Fokus, Tastaturbedienung.
- Varianten über Inputs steuern, nicht über kopierte Komponenten.
- Stories/Beispiele und Visual-Tests für jede öffentliche Komponente.
